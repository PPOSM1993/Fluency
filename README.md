## 📅 Calendario Tentativo de Desarrollo (8 semanas)

## 🔹 Semana 1 – Configuración inicial (backend)

```bash
- Crear proyecto Django + configuración de Postgres.
- Modularizar settings (base/dev/prod).
- App accounts/ con registro + login JWT.
- Probar endpoints de autenticación con Thunder Client/Postman.
- 👉 Entrega: Backend con login/registro funcional y DB conectada.
```

## 🔹 Semana 2 – Chat básico con IA

```bash
- App chat/: modelos Conversation y Message.
- Endpoint /api/chat/ conectado a OpenAI API.\
- Guardar historial en la base de datos.
- Testear conversación básica (texto).
- 👉 Entrega: Puedes conversar con la IA y guardar historial.
```

## 🔹 Semana 3 – Frontend inicial (Next.js + Tailwind)

```bash
- Crear proyecto Next.js.
- Configurar TailwindCSS + shadcn/ui.
- Pantallas de login/register conectadas al backend.
- Pantalla de chat con UI básica.
- 👉 Entrega: Web app con login y chat funcional.
```

## 🔹 Semana 4 – Mejoras en el chat

```bash
- Mostrar historial de conversaciones.
- Mejorar la UI (ChatMessage, ChatBox).
- Corrección gramatical en tiempo real (prompt engineering o LanguageTool).
- 👉 Entrega: Chat con historial y correcciones automáticas.
```

## 🔹 Semana 5 – Lecciones y vocabulario

```bash
- App lessons/ en backend (estructura inicial).
- API para lecciones/vocabulario.
- Frontend: pantalla de lecciones y ejercicios básicos.
- 👉 Entrega: Sistema de lecciones conectadas a la IA.
```

## 🔹 Semana 6 – Seguimiento de progreso
```bash
- App progress/ en backend (estadísticas de usuario).
- Guardar métricas: número de palabras, correcciones, tiempo de práctica.
- Frontend: dashboard de progreso básico.
- 👉 Entrega: Dashboard de progreso funcionando.
```

## 🔹 Semana 7 – Voz y experiencia avanzada

```bash
- Integrar Web Speech API (voz a texto y TTS).
- Chat por voz en el frontend.
- Guardar transcripciones en backend.
- 👉 Entrega: Chat con voz funcionando.
```
## 🔹 Semana 8 – Pulido y despliegue

```bash
- Pulir UI (estilos finales).
- Configurar Docker + docker-compose para levantar todo junto.
- Deploy en Vercel (frontend) y Railway/Render (backend).
- Documentación final en README.

👉 Entrega: Versión MVP lista en producción 🚀.
```


## 🧭 Resumen

```bash
- Mes 1: Backend + frontend básico con chat e historial.
- Mes 2: Lecciones, progreso, voz y despliegue final.
```
